## Installation

1. Install dependencies

```bash
$ yarn install
```

2.Run docker with migation

```bash
$ yarn db:dev:restart
```

4. Run the app


## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# e2e tests
$ yarn test:e2e
```
